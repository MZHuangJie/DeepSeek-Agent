关键设计决策

  • Activity Bar（最左 44px）：切换主侧边栏内容（文件/搜索/Agent/设置），类似 VS Code 的图标栏
  • 编辑器+聊天上下分屏：上方代码编辑器，下方聊天面板，可拖拽调整比例。编辑器独占时聊天收缩到底部
  • Agent 可观测面板（右侧 280px）：独立展示思考链、工具调用时间线、Token 统计。可折叠
  • 底部终端：集成终端，Agent 执行的命令在此显示
