# Oh My DeepSeek

桌面端 AI 助手，集成了多种大模型，提供代码编写、项目管理、AI 对话、角色扮演、云端同步等功能。

## 功能

### AI 对话
- **五种模式**：编程助手、聊天、角色扮演、计划模式、Multi-Agent 协作
- 流式输出，实时显示思考过程和工具调用
- 命令系统：`/plan`、`/review`、`/brainstorming`、`/debugging` 等
- Markdown 渲染，支持代码高亮、表格、图片
- 粘贴图片到对话，自动识别并描述
- 会话标题自动生成

### Agent 工具
- 文件操作：读取、写入、编辑、搜索代码
- 终端执行：内置多实例终端（PowerShell/cmd/bash）
- 子代理：大型项目自动拆分任务并行处理
- 生图：调用多种生图模型生成图片
- 网页预览：生成交互式 HTML 页面并实时预览
- 网络搜索、网页抓取、网页截图
- 图片理解：调用视觉模型描述图片

### 文件管理
- 文件树浏览器，支持创建、重命名、删除
- 右键菜单：新建、重命名、删除、添加到对话
- 打开文件/文件夹（支持单个文件打开，自动设为工作区）
- 记住上次打开的文件夹
- 最近打开的工作区列表

### 代码编辑器
- Monaco Editor，多语言语法高亮
- 图片查看器
- 未保存状态指示

### 终端
- 真实终端模拟（node-pty）
- 多 Tab 切换
- 切换工作区自动 cd

### 角色系统
- 创建、编辑、删除角色，基于模板创建
- 预设模板（产品经理、设计师、前后端工程师、QA 等）
- 状态面板：自定义状态字段（数值/文本/列表）
- AI 生成立绘：多种画风可选，支持参考图
- 角色扮演聊天：单人沉浸式 + 群像多人模式
- 立绘上传到云端

### 账户中心 & 角色广场
- 账户注册/登录
- 浏览、搜索、收藏其他用户分享的角色
- 一键恢复到本地
- 角色和模板云端同步

### Multi-Agent 群聊
- 创建 Agent 群聊，配置多个角色
- @ 点名指定成员回复，不 @ 则全员参与
- 每个角色有明确的职责边界
- 支持 Markdown 格式回复

### 模型管理
- 多模型切换，自定义 API Key / Base URL
- 支持 OpenAI、Anthropic、DeepSeek、通义千问、智谱、Gemini 等
- 生图模型独立配置
- 视觉模型独立配置

### 界面
- 左侧 Activity Bar 切换面板
- 右侧 Agent 观测面板（工具调用、Token 用量、余额查询）
- 深色主题

## 使用

### 首次启动
1. 打开软件后，点击左侧面板的"打开文件夹"选择项目目录
2. 在聊天面板底部输入 API Key
3. 在 ⚙ 设置中添加和管理模型

### 基本操作
- 输入框输入消息，Enter 发送，Shift+Enter 换行
- `/` 触发命令面板
- `@` 触发文件引用或群聊成员选择
- 粘贴图片到输入框
- Ctrl+S 保存当前编辑的文件

### 群聊
1. 左侧切换到会话面板，点击底部的"NPC 群聊"或"Agent 群聊"
2. 勾选参与成员（至少 2 人）
3. 创建后，输入 `@成员名` 指定回复对象，不 @ 则全员讨论

### 云同步
1. 注册/登录账户
2. 在角色面板点击 ☁ 同步角色和模板
3. 会话自动同步到云端
4. 换电脑后从云端一键恢复
